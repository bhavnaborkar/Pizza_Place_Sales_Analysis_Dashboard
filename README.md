# Pizza_Place_Sales_Analysis_Dashboard
Analyzed $817.86K in sales from 21K orders and 50K pizzas sold, deriving key KPIs such as average order size (2.28 pizzas/order) using DAX. 
Identified monthly sales trends, with peak monthly revenue reaching $73K and seasonal fluctuations across the year. 
Designed an interactive Power BI dashboard using line, bar, donut, stacked charts, slicers, and AIpowered Q&amp;A visual to uncover monthly trends, size &amp; category performance, and top-selling products for data-driven decisions.
